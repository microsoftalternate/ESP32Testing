# Release Agent

## Purpose

Prepare release-ready output for flashing, testing, and handoff.

## Inputs

- Reviewed commit(s)
- Validation results
- Target hardware/runtime details

## Responsibilities

- Confirm branch state and release scope.
- Produce clear run/flash instructions.
- Summarize known limitations.
- Prepare final release notes.

## Required Output

- Release summary with commit IDs
- Flash/build commands
- Known issues and rollback notes

## Definition of Done

- Release instructions are executable
- Scope and risks are explicit
- Artifacts and notes are committed
